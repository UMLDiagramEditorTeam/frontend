# PROJECT 

Веб-приложение для разработчиков - интерактивный графический редактор, позволяющий упростить переход от проектного моделирования к написанию кода за счет автоматической генерации программных заготовок (скелетонов)

Приложение предоставляет пользователям пространство для создания, сохранения и редактирования UML-диаграмм через drag-and-drop интерфейс

Ключевой особенностью является механика генерации кода: при создании диаграммы классов система автоматически формирует структуру файлов на языках Python или Java, включая описание классов, атрибутов, методов и модификаторов доступа. Для диаграмм последовательностей приложение генерирует скелеты методов с соответствующей логикой вызовов между объектами.

# Запуск проекта

## Для разработки

```
npm install
npm run dev
```

## Для сборки проекта

```
npm install
npm run build
```
Для прода локально:
`npm run preview`

# Форматирование кода

Проверка кода линтером:
`npm run lint`

Запуск форматирования кода:
`npm run format`

# Git Hooks

Для включения pre-commit хуков:
`npm run prepare`

## Для генерации API клиента 

Используем автоматическую генерацию типов и SDK клиента на основе OpenAPI спецификации бэкенда

Для генерации используется версия спеки, размещенная на SwaggerHub

Чтобы запустить генерацию:
`npm run generate:api`
